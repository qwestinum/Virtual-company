Sauvegarde l'état actuel :

1. Liste tous les fichiers modifiés depuis le dernier commit
2. Résume ce qui a été fait dans cette session
3. Identifie ce qui reste à faire (référence SPEC.md)
4. Écris un résumé dans docs/PROGRESS.md
5. Fais un commit avec un message conventionnel descriptif
